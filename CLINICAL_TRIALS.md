## Active and Relevant Trials

### 1. NCT04725461 — Low Cost Socket for Lower Limb Amputees
| Field | Details |
|-------|---------|
| **Status** | COMPLETED |
| **Sponsor** | Shirley Ryan AbilityLab (Chicago, IL) |
| **PI** | Laura Miller |
| **Focus** | Affordable prosthetic socket fabrication using expandable rigid foam |
| **Innovation** | 30-min casting, 90-min socket completion — suitable for resource-limited settings |
| **Location** | 355 E Erie St, Chicago, IL 60611 |
| **Enrollment** | 8 transtibial amputees (K2-K4) |
| **Key Outcome** | Socket Comfort Score vs. conventional socket; functional mobility measures |

#### Relevance to Outreach Initiative
- Directly addresses affordable prosthetic access in developing countries
- Proven simplified method that can be replicated in low-resource settings
- Chicago-based — serves as the primary hub for prosthetic technical expertise
- Patent-pending fabrication method

## Contributing New Trials
To add a trial to this database:
1. Search ClinicalTrials.gov for the trial
2. Add the NCT ID and key details to this file
3. Create a corresponding Issue in the GitHub tracker (if it doesn't exist already)
4. Tag it with the `clinical-trial` label
