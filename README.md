#### NCT04725461 — Low Cost Socket for Lower Limb Amputees
- **Sponsor:** Shirley Ryan AbilityLab
- **Principal Investigator:** Laura Miller
- **Description:** Testing an investigational fabrication system for transtibial prostheses using expandable rigid foam. The goal is to provide affordable, high-quality socket fitting for amputees in developing countries. The method allows casting in <30 min and socket completion in 90 min.
- **Key Innovation:** Simplified socket fitting protocol using expandable rigid foam (Foam IT 8) — eliminates the need for multi-step manual fabrication, reducing errors and costs.
- **Trial Location:** Shirley Ryan AbilityLab, 355 E Erie St, Chicago, IL 60611
- **Enrollment:** 8 participants (unilateral transtibial amputees, K2-K4 level)
